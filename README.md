# img/ — Mapa za slike

Ova mapa sadrži sve slike i medijske datoteke projekta.

## Potrebne datoteke

| Datoteka | Izvorna datoteka | Format | Opis |
|---|---|---|---|
| `niko-juricic.jpg` | `niko-juricic.png` | JPG / PNG | Hero fotografija na početnoj stranici |
| `niko-juricic-about.jpg` | `niko-juricic-about.png` | JPG / PNG | Fotografija na O meni stranici |
| `project-weatherapp.png` | — | PNG | Screenshot WeatherApp projekta |
| `project-todo.png` | — | PNG | Screenshot Todo Manager projekta |
| `project-blog.png` | — | PNG | Screenshot Blog Template projekta |
| `project-kalkulator.png` | — | PNG | Screenshot Kalkulator projekta |
| `og-image.jpg` | — | JPG | Open Graph slika (dijeljenje na društvenim mrežama) |
| `cv-niko-juricic.pdf` | — | PDF | Životopis za preuzimanje |

## Savjeti za optimizaciju

- Komprimiraj JPG slike na ~80% kvalitete (alat: [Squoosh](https://squoosh.app))
- Koristi `loading="lazy"` atribut za slike ispod fold-a
- Uvijek navedi `width` i `height` atribute na `<img>` tagovima za sprječavanje layout shift-a
- Za screenshote projekata: napravi ih u pregledniku na rezoluciji 1280×720, zatim smanji na 800×450
